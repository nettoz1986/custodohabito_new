# Importacao do projeto "Custo do Habito" vindo do ChatGPT

Use esta pasta para concentrar tudo o que vier do seu projeto no ChatGPT e que deve influenciar o agente deste app.

## Ordem sugerida

1. Preencha `project-overview.md` com a visao central do projeto.
2. Preencha `voice-and-tone.md` com o jeito de falar, limites e frases-chave.
3. Preencha `core-concepts.md` com conceitos, teses e ideias estruturais.
4. Preencha `conversation-patterns.md` com exemplos de boas respostas, tipos de conversa e perguntas que o agente deve fazer.
5. Se houver materiais longos, crie arquivos extras aqui mesmo:
   - `offers.md`
   - `personas.md`
   - `diagnostic-logic.md`
   - `objections.md`
   - `faq.md`

## Como colar o conteudo

- Pode copiar e colar diretamente do ChatGPT para os arquivos `.md`.
- Nao precisa limpar tudo antes. Eu consigo te ajudar a organizar depois.
- Se uma conversa estiver grande, crie um arquivo como `chat-export-01.md`, `chat-export-02.md`.

## O que ajuda mais

- Tese central do projeto
- Promessa de transformacao
- O que o projeto evita ou critica
- Tipos de usuario e perfis
- Vocabulos e expressoes do projeto
- Exemplos de respostas que "soam certas"
- Exemplos de respostas que "nao parecem Custo do Habito"
- Decisoes praticas que o agente deve saber analisar
- Padroes emocionais e comportamentais que o agente deve saber nomear

## Depois

Quando voce preencher isso, eu consigo:

- mapear o que entra em cada arquivo do app
- comparar o contexto novo com o que ja existe
- propor a ordem ideal de implementacao
- editar um modulo por vez com seguranca
